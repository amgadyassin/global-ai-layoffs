# Global AI Layoffs & Job Market Analysis (2020–Present)

Exploratory data analysis and interactive dashboard on global AI industry layoffs.

**Dataset:** [Kaggle – Global AI Layoffs 2020–Present](https://www.kaggle.com/datasets/belbino/global-ai-layoffs-and-job-market-2020-present)

## Notebooks
- `notebooks/01_data_cleaning.ipynb` — Data cleaning & preprocessing
- `notebooks/02_data_visualization.ipynb` — Charts & EDA
- `notebooks/03_dashboard.ipynb` — Interactive Dash + Plotly Express dashboard

## Stack
Python | Pandas | Plotly Express | Dash | Jupyter
